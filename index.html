Ressources pour les développeurs JavaScript
Back-end, Front-end et Full-Stack.

Ce site fourni des ressources, certaines en libre accès, sur l'apprentissage de la programmation d'applications centrées sur l'éco-système JavaScript. Il est le fruit de plusieurs années pour l'auteur d'enseignements en informatique et encore maintenant pour des étudiants en MASTER.

Cadre de travail

Force est de constater que beaucoup de développeurs, à l'exception du cas particulier des MAC, préfèrent comme cadre de travail un système Windows. Si la mise en production reste l'apanage de Linux, le développeur peut toutefois aisemment utiliser ses logiciels préférés sur un poste de travail Windows.

Les deux situations seront combinées par l'utilisation d'une machine virtuelle de façon à disposer des deux environnements : développement et production

Bases indispensables
Au niveau matériel, il faut au développeur au minimum un PC (de préférence portable), avec 8Go de RAM, d'un processeur de type Core I5 et d'un disque >= 512 Go.
Suit ci-dessous la liste des logiciels à installer pour le développeur WEB ; le parti pris est que, pour être dans l'esprit Linux et dans la mouvance Open Source, tous ces logiciels disposent d'une licence permettant une utilisation gratuite.
  •   Git - Logiciel pour le versionning
    Info : Choisissez GitLab ou GitHub de préférence pour le service de dépôt.
  •   Chrome - Navigateur Internet de tests
    Info : Utiliser Chrome uniquement en localhost pour les applications WEB.
  •   Visual Studio Code - IDE
    Info : Le meilleur pour le développement JavaScript.
  •   Node.js - Framework JavaScrip Back-end
    Info : Le framework de base avec NPM pour le développement JavaScript.
  •   VirtualBox - Logiciel de virtualisation
    Info : Plus simple d'utilisation que VMware Player.
  •   Lubuntu - Distribution Linux "light"
    Info : Distribution la moins gourmande en ressources pour une virtualisation.
  •   MongoDB - SGBD orientée documents
    Info : SGBDD idéale pour la mise en place d'API.
